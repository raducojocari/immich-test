## Context
I want to be able to host on my local network a goodle photos alternative such that I can keep all my photos secure. Projects like Immich allow you to do just that.
Because I want to keep my photos safe, I am storing them on a NAS from Unifi. 

## Requirements
1. Build a script that will pull down the immich repo and deploy it locally in docker containers. Install it by following the install scripts. The installation will happen on a macbook. The github repo is here: https://github.com/immich-app/immich. the install script will be called install/sh and it will be stored in a dir called ./output.
2. Build a script that allows me to start and stop immich. the script should created in ./output. the script should be called ./start.sh and ./stop.sh
3. The build script should configure the immich containers to read the photos from '/Volumes/nas/Google Photos/Radu'.
4. The immich storage should be stored on the NAS under a new dir called immich

## Non-Functional requirements
1. Proper error handling: The scripts should have proper error handling. pre-requisits check. Check the local machine for any software that needs to be present before running any scripts and error with suggestive error message.
2. Readability: The scripts code should be easy to read
3. Maintainability: The script should be easy to change. Do not deviate much from the standard immich config in order to allow for a maintainable script.
